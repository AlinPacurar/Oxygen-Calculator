Oxygen Calculator

A cleaner public-facing calculator inspired by the structure of modern online oxygen tools: prominent disclaimer, straightforward inputs, large result cards, formula transparency, cylinder support, and quick validation against the spreadsheet defaults.

Clinical disclaimer

This tool is intended as a planning and educational aid only. It is not a substitute for clinical judgement, local policy, device-specific instructions for use, or bedside verification. Always confirm ventilator settings, oxygen source availability, cylinder content, transport duration, reserve requirements, and contingency before clinical use.

Deployment note

This build is a static HTML app intended for GitHub Pages. Upload the files to a repository root, enable Pages for the main branch, and the calculator will be reachable from any device with a browser.

This browser build is designed to resemble a modern online calculator pattern, while remaining your own standalone implementation. Re-validate all formulas against the source spreadsheet before live use.
